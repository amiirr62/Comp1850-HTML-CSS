Web Analysis

1. Competitor Analysis

Competitor Site 1: Sortly – Inventory Management

  • Clean, minimalistic interface with bold icons
  • Clear CTAs (Call to Action) for adding/viewing inventory
  • Mobile-first responsive design

Competitor Site 2: Zoho Inventory

  • Professional layout with well-organized navigation
  • Dashboard-style widgets for quick insights
  • Consistent color coding for different inventory statuses

2. A Site I Use Regularly

Website: Canada.ca

This government site is highly usable and functional. Its main objectives are to provide easy access to official information and digital services. Two design features that help it succeed:

  • Consistent headers and accessibility-friendly design
  • Well-labeled content sections with clear navigation and search

3. Site with Effective Use of Color

Website: Asana

Asana uses a balanced pastel palette with bold accents:

  • Main colors: #273c75 (deep blue), #fbc531 (sun yellow), #00a8ff (sky blue)
  • These colors create a sense of clarity and calm while directing user attention
I may use similar colors for alert messages or category tags in my project.

4. Site with an Appealing Layout

Website: Slack

Slack’s layout feels dynamic and user-friendly:

  • Smart use of grids, full-width hero sections, and card layouts
  • Consistent spacing and typography for readability
This kind of visual structure inspired how I organized inventory tables and filter sections in my project.
